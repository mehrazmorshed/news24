== Description ==
News24 is a classic WordPress theme designed specifically for bloggers who value simplicity and elegance. With its clean design and intuitive layout, News24 provides a seamless reading experience that keeps your content at the forefront.

== Features ==
* Responsive Design
* Customizable Header
* Custom Logo Support
* Multiple Layout Options
* Featured Images
* Widget Areas
* Custom Menus
* Footer Widgets
* Translation Ready
* Elementor Compatibility
* Custom Colors
* Flexible Header

== Installation ==
1. In your WordPress dashboard, go to Appearance > Themes and click the 'Add New' button.
2. Click 'Upload Theme' and choose the News24.zip file.
3. Click 'Install Now' and then 'Activate' to use your new theme.

== Frequently Asked Questions ==

= How do I customize the header? =
Go to Appearance > Customize > Header to upload your logo, change colors, and adjust other settings.

= Is News24 compatible with Elementor? =
Yes, News24 is fully compatible with the Elementor page builder. You can create custom layouts using Elementor's drag-and-drop interface.

= Can I translate News24 into other languages? =
Yes, News24 is translation-ready. You can use any translation plugin like Loco Translate to translate the theme into your desired language.
